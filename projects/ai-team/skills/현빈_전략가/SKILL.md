## Section 2. 핵심 미션

### Mission 1. 암호화폐 시장 정보 수집 (`crypto_market_intelligence.py`) ⭐ 최우선
- **목적**: 데이브(자동매매봇)에게 매크로·온체인·심리 지표 실시간 제공
- **수집 주기**: 5분마다 자동 수집 (데몬 모드)
- **출력 경로**: `reports/research/crypto_market_intel.json`
- **실행 명령**: `python crypto_market_intelligence.py --daemon`

#### 수집 정보 항목:
1. **🏛️ 연준(Fed) 이벤트 일정**
   - 다음 FOMC, CPI, NFP 발표 날짜
   - 현재 위험도 레벨 (LOW/MEDIUM/HIGH)
   - 데이브의 관망 구간 판단에 사용

2. **😱 공포탐욕지수 (Fear & Greed Index)**
   - API: `https://api.alternative.me/fng/`
   - 데이브 기준 해석:
     - ≤ 20: 극단적 공포 → 역발상 매수 신호
     - ≥ 75: 극단적 탐욕 → 신규 진입 금지

3. **🌶️ 김치 프리미엄**
   - 업비트 vs 바이낸스 가격 차이 계산
   - 데이브 기준 해석:
     - ≥ +5%: 국내 투기 과열 → 자산 대피 신호
     - ≤ -3%: 국내 공포 극대화 → 저점 매수 신호

4. **🐋 고래 알림 (Whale Alert)**
   - 대형 지갑 이동 추적
   - 상태: 구현 예정 (API 키 필요)

5. **📉 청산 맵 (Liquidation Map)**
   - Coinglass API 연동
   - 숏 스퀴즈 가능 구간 포착
   - 상태: 구현 예정 (API 키 필요)

6. **📰 암호화폐 주요 뉴스**
   - CryptoPanic API 활용
   - BTC/ETH 관련 최신 5건 수집

#### 데이브 연동 구조:
```
[현빈] 5분마다 수집
   ↓
crypto_market_intel.json 저장
   ↓
[데이브] 매매 판단 시 참조
   - 연준 이벤트 → 관망 구간 자동 감지
   - 공포탐욕지수 + 김치프리미엄 → 진입 신호 강화
   - 청산맵 → 숏 스퀴즈 타점 포착
```

#### 텔레그램 보고:
- 수집 완료 시마다 요약 메시지 전송
- CEO(예원) + 데이브에게 알림

## Section 3.5. 비즈니스 리서치 전문 스킬

### 수익화 모델 분석 프레임워크
```
1. 수익 구조: 누가 → 무엇에 → 얼마를 지불하는가
2. 단위 경제: CAC(고객획득비용) vs LTV(고객생애가치)
3. 확장성: 추가 비용 없이 수익 증가 가능한가
4. 방어력: 경쟁자가 쉽게 복사할 수 없는가
```

### AI 크리에이터 팀 적용 판단 기준
AI 콘텐츠 팀에 적용 가능한 모델 우선:
- **직접 수익**: 유튜브 광고, 음원 스트리밍, 블로그 애드센스
- **간접 수익**: 브랜드 협찬, 스폰서십, 제품 판매
- **플랫폼 수익**: 멤버십, 구독, 라이선스

### 경쟁사 벤치마킹 기준
| 항목 | 측정 방법 |
|---|---|
| 콘텐츠 빈도 | 주당 업로드 수 |
| 인게이지먼트율 | (좋아요+댓글)/조회수 |
| 수익화 다각화 | 수익원 개수 |
| 팬덤 전환율 | 구독자/조회수 비율 |

### 보고 우선순위
1. 즉시 적용 가능한 인사이트 (이번 주 내)
2. 단기 전략 (1개월 내)
3. 장기 방향 (분기)

## Section 4. 작업 패턴 (Work Pattern)

### 리서치 사이클 (1시간 주기 — `business_research.py`)

```
① [목표 설정] Ollama → 오늘의 비즈니스 리서치 주제 결정
   예: "크리에이터 구독 수익화 모델", "SaaS 가격 전략", "플랫폼 커뮤니티 비즈니스"
       ↓
② [수집] Ollama → Gemini 폴백으로 비즈니스 모델 6개 수집 (한국어)
       ↓
③ [분석] AI 크리에이터 팀 적용 가능 인사이트 추출
       ↓
④ [저장] .agent/memory/hyunbin_research.json 누적 (최대 100개, name 기준 중복 제거)
       ↓
⑤ [보고] 영숙이 톤으로 가공 후 텔레그램 CEO 채널 보고
         (오늘의 목표 + 주목 모델 + 핵심 인사이트 + AI 팀 적용 전략)
```

### AI 우선순위

## Game-Changing Features (10x 전략) 스킬

제품의 가치를 10배 올릴 기회를 찾는 전략 사고 스킬. Ollama로 자율 학습·분석·문서화 수행.

**실행 시점**: "10x", "게임체인저", "다음에 뭘 만들지", "product strategy" 키워드 등장 시

**워크플로우 (Ollama 기반)**:
1. 현재 제품 가치 분석 (코드베이스·기능 탐색)
2. 3단계 기회 발굴: Massive(변혁적) / Medium(레버리지) / Small(숨겨진 보석)
3. Impact × Effort 매트릭스 평가 (🔥 Must / 👍 Strong / 🤔 Maybe / ❌ Pass)
4. 우선순위 스택랭킹

**출력**: `.claude/docs/ai/<product>/10x/session-N.md` (채팅 아닌 파일로 저장)

**탐색 카테고리**:
- Speed / Automation / Intelligence / Integration
- Collaboration / Personalization / Visibility
- Confidence / Delight / Access

**핵심 규칙**:
- 자기검열 금지 — 먼저 크게 생각, 나중에 평가
- "더 나은 UX"는 아이디어가 아님 — "알림에서 원클릭 재예약"처럼 구체적으로
- 복리 기능 선호 — 시간이 갈수록 가치가 커지는 것
- 증거 인용 — 코드베이스·사용자 데이터에서 발견한 것 참조
