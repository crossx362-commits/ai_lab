# 머신러닝

## 핵심 개념

*   **지도 학습 (Supervised Learning):** 레이블된 데이터를 활용하여 예측 모델 구축 (분류, 회귀).
*   **비지도 학습 (Unsupervised Learning):** 레이블 없는 데이터에서 패턴 발견 (군집화, 차원 축소).
*   **이상 탐지 (Anomaly Detection):** 정상 데이터 패턴에서 벗어난 비정상 데이터 식별.
*   **피처 엔지니어링 (Feature Engineering):** 모델 성능 향상을 위한 데이터 전처리 및 특징 생성.
*   **모델 평가 (Model Evaluation):** 정확도, 정밀도, 재현율, F1-score 등을 활용하여 모델 성능 측정.

## 실전 적용

*   **로그 분석:** 머신러닝 기반 분류 모델을 사용하여 악성 로그 자동 분류 (예: 악성코드 감염 시도, 무단 접근).
*   **네트워크 트래픽 분석:** 이상 탐지 알고리즘을 활용하여 비정상적인 네트워크 트래픽 패턴 식별 (예: DDoS 공격, 데이터 유출).
*   **사용자 행동 분석:** 사용자 행동 패턴을 학습하여 비정상적인 활동 탐지 (예: 계정 탈취, 내부자 위협).
*   **피처 엔지니어링 예시:** IP 주소의 국가 코드, 사용 시간 등 특징을 생성하여 모델 정확도 향상.

## 경수에게 유용한 이유

보안 분석, 리스크 탐지, 이상행동 모니터링 업무에서 머신러닝은 자동화된 분석 및 예측을 가능하게 합니다. 알려지지 않은 위협을 탐지하고, 이벤트의 우선순위를 결정하며, 대응 시간을 단축하는 데 필수적인 스킬입니다. 특히 이상 탐지 능력은 새로운 공격 패턴에 대한 민감도를 높여줍니다.
