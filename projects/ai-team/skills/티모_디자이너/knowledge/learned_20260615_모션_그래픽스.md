# 모션 그래픽스

## 핵심 개념
1. **타이밍과 이징(Easing):** 선형적 움직임이 아닌 가속/감속을 활용해 물리 법칙에 기반한 자연스러운 움직임을 구현합니다.
2. **연속성 및 흐름:** 요소가 끊기지 않고 연결되는 경로를 설계하여 시각적 연속성을 확보합니다.
3. **시각적 위계(Hierarchy):** 모션의 크기와 속도 대비를 통해 사용자의 시선을 의도한 곳에 집중시킵니다.
4. **상호작용 피드백:** 사용자 액션에 즉각적인 반응을 제공하여 시스템 상태를 직관적으로 전달합니다.

## 실전 적용
- **마이크로 인터랙션:** 버튼 클릭, 스위치 전환 시 생동감 있는 애니메이션을 부여해 조작감을 개선합니다.
- **화면 전환(Transition):** 페이지 이동 시 컨텐츠가 유기적으로 연결되어 공간적 흐름을 매끄럽게 만듭니다.
- **브랜딩 애니메이션:** 로고나 그래픽에 브랜드 아이덴티티를 담은 고유한 움직임 규칙을 설정합니다.

## 티모에게 유용한 이유
UI/UX 디자이너로서 정적인 화면에 '시간'과 '생동감'을 더해 사용자 경험의 완성도를 높입니다. 시각 브랜딩 측면에서 일관된 모션 가이드를 수립함으로써 브랜드 고유의 감성을 강화하고, 타이포그래피와 컬러를 동적으로 표현하여 더욱 강력한 비주얼 메시지를 전달할 수 있습니다.
