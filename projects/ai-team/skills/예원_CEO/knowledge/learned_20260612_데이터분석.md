# 데이터분석

## 핵심 개념

* **데이터 분석 프로세스:** 문제 정의 → 데이터 수집/정제 → 탐색적 데이터 분석(EDA) → 모델링 → 평가/결론 도출
* **가설 검정:** 통계적 유의성을 기반으로 가설의 타당성 검증 (p-value, 신뢰구간 이해)
* **시각화:** 데이터 패턴 발견 및 효과적인 전달을 위한 차트/그래프 활용 (막대, 선, 산점도 등)
* **회귀 분석:** 변수 간 관계 모델링 및 예측 (선형 회귀, 로지스틱 회귀 등)
* **A/B 테스트:** 두 가지 방법의 효과 비교 (통계적 유의성 중요)

## 실전 적용

* **OKR 설정:** 데이터 기반으로 OKR 달성 가능성 예측 및 조정 (예: 전환율 데이터 분석 기반 OKR 목표 설정)
* **팀 성과 측정:** 데이터 대시보드를 활용하여 팀 성과 실시간 모니터링 및 개선점 파악
* **의사 결정 지원:** A/B 테스트 결과 분석을 통해 최적의 전략 선택 (예: 광고 캠페인 효율 분석)
* **문제 해결:** 데이터 분석을 통해 숨겨진 문제점 발견 및 해결 방안 도출 (예: 고객 이탈 원인 분석)

## 예원에게 유용한 이유

전략적 의사결정, 팀 리더십, 비전 수립 과정에서 데이터 기반의 객관적인 근거 마련 가능. OKR 설정 및 성과 측정 시 데이터 분석 스킬은 필수적이며, 팀의 데이터 리터러시 향상에 기여.
