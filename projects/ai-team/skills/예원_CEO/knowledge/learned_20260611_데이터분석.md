# 데이터분석

## 핵심 개념

* **데이터 분석 프로세스:** 문제 정의 -> 데이터 수집/정제 -> 탐색적 데이터 분석(EDA) -> 모델링 -> 평가/해석 -> 의사결정
* **기술 통계:** 평균, 중앙값, 표준편차 등 데이터의 분포와 특징 파악
* **시각화:** 막대 그래프, 산점도, 히스토그램 등을 활용하여 데이터 패턴 인지
* **가설 검정:** 통계적 유의성을 검증하여 결론 도출
* **회귀 분석:** 변수 간 관계 모델링 및 예측

## 실전 적용

* **문제 정의:** "A 제품 판매량 감소 원인 분석" -> 데이터 기반 가설 설정 (예: 경쟁사 프로모션, 계절적 요인)
* **EDA:** 판매량, 마케팅 지출, 경쟁사 프로모션 데이터 시각화하여 패턴 발견
* **가설 검정:** 마케팅 지출 증가가 판매량에 유의미한 영향을 미치는지 t-test 수행
* **회귀 분석:** 과거 판매 데이터 기반 판매량 예측 모델 구축

## 예원에게 유용한 이유

* **전략적 의사결정:** 데이터 기반 객관적 근거 제시, 의사결정 성공률 향상
* **팀 리더십:** 분석 결과 명확하게 전달, 팀원들의 데이터 리터러시 향상
* **비전 수립:** OKR 설정 및 달성 현황 측정 지표 개발, 데이터 기반 목표 관리
* **OKR:** OKR 목표 달성 여부를 데이터 분석으로 측정하고, 개선 방안 도출
