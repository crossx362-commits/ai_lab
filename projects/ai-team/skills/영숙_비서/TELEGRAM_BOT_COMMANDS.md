## 🤖 봇 제어 명령어

### 봇 상태 확인
```
봇상태
bot status
```
- 윈도우/맥북의 모든 봇 상태 조회
- 실행 중인 봇의 PID 표시

### 모든 봇 종료 (재시작 안 함)
```
봇전체종료
봇모두종료
stopallbots
```
- **중요**: 윈도우와 맥북의 **모든 봇**을 종료
- 자동 재시작 **안 됨**
- 외출할 때 또는 완전히 종료하고 싶을 때 사용

### 모든 봇 시작
```
봇전체시작
봇모두시작
startallbots
```
- 윈도우와 맥북의 모든 봇 시작 (SSH 설정 필요)
- 집에 돌아왔을 때 사용

### 로컬 봇만 종료
```
봇종료
bot stop
```
- 현재 명령을 받은 봇만 종료
- 맥북에서 명령 → 맥북 봇만 종료
- 윈도우에서 명령 → 윈도우 봇만 종료

### 로컬 봇만 시작
```
봇시작
bot start
```
- 현재 머신의 봇만 시작

### 맥북 봇 원격 제어 (SSH 필요)
```
맥북봇종료
macbook stop

맥북봇시작
macbook start
```
- SSH가 설정되어 있어야 작동
- 윈도우에서 맥북 봇을 원격으로 제어

## 👥 에이전트 제어 명령어

### 전체 에이전트 상태
```
에이전트상태
agent status
```
- 데이브, 레오, 펄스, 영숙 모든 에이전트 상태
- 실행 중인 에이전트는 🟢, 중지는 🔴 표시

### 특정 에이전트 시작
```
데이브 시작
레오 시작
펄스 시작

dave start
leo start
```
- 해당 에이전트만 시작
- 이미 실행 중이면 경고 메시지

### 특정 에이전트 종료
```
데이브 종료
레오 종료
펄스 종료

dave stop
leo stop
```
- 해당 에이전트만 종료
- 실행 중이 아니면 경고 메시지

### 특정 에이전트 재시작
```
데이브 재시작
레오 재시작

dave restart
leo restart
```
- 종료 → 2초 대기 → 시작

### 특정 에이전트 상태 확인
```
데이브 상태
레오 상태

dave status
leo status
```
- 개별 에이전트의 상태와 PID 조회

**사용 가능한 에이전트:**
- 데이브 (dave) - 보수적 트레이더
- 레오 (leo) - 공격적 트레이더
- 펄스 (pulse) - 시장 정보 수집가
- 영숙 (youngsuk) - 텔레그램 봇

## 💡 사용 시나리오

### 시나리오 1: 외출할 때
```
1. "봇전체종료" 전송
   → 맥북과 윈도우의 모든 봇 종료 (전력 절약)

2. 필요하면 특정 에이전트만 재시작
   → "펄스 시작" (시장 정보만 계속 수집)
```

### 시나리오 2: 집에 돌아올 때
```
1. "봇전체시작" 전송
   → 모든 봇과 에이전트 시작

또는 개별적으로:
2. "봇시작" → 텔레그램 봇만 시작
3. "데이브 시작" → 트레이딩 시작
4. "레오 시작" → 공격적 트레이딩 시작
```

### 시나리오 3: 트레이딩만 일시 중지
```
"데이브 종료"
"레오 종료"
→ 트레이더만 종료, 봇과 정보 수집은 계속
```

### 시나리오 4: 409 Conflict 해결
```
상황: 맥북과 윈도우 봇이 동시 실행되어 충돌

해결:
1. "맥북봇종료" (SSH 설정되어 있으면)
   또는
2. "봇전체종료" → 5초 대기 → "봇시작"
```

## 🔧 고급 기능

### SSH 설정 (맥북 원격 제어용)

1. `.env` 파일에 추가:
```env
MACBOOK_SSH_HOST="사용자명@맥북IP"
MACBOOK_SSH_KEY_PATH="C:/Users/User/.ssh/macbook_key"
```

2. SSH 키 생성:
```bash
ssh-keygen -t ed25519 -f ~/.ssh/macbook_key
```

3. 맥북에 키 복사:
```bash
ssh-copy-id -i ~/.ssh/macbook_key.pub user@macbook-ip
```

자세한 설정은 `REMOTE_CONTROL_GUIDE.md` 참고

## ⚠️ 주의사항

1. **"봇전체종료"는 신중하게 사용**
   - 모든 봇이 종료되고 자동 재시작 안 됨
   - 다시 시작하려면 "봇전체시작" 또는 수동 시작 필요

2. **에이전트 시작 순서**
   - 펄스 (시장 정보) → 데이브/레오 (트레이더) 순서 권장
   - 트레이더는 펄스의 정보에 의존

3. **동시 실행 금지**
   - 맥북과 윈도우에서 **같은 봇을 동시에 실행하면 안 됨**
   - 409 Conflict 에러 발생

4. **명령어는 대소문자 구분 안 함**
   - "데이브 시작" = "데이브 시작" = "dave start"
   - 띄어쓰기도 자유롭게

## 🆘 문제 해결

### 봇이 응답하지 않을 때
1. "봇상태" 전송해서 실행 중인지 확인
2. 실행 중이 아니면 "봇시작"
3. 여전히 안 되면 PowerShell에서 직접 시작:
   ```powershell
   python D:\ai_lab\projects\ai-team\skills\영숙_비서\tools\telegram_receiver.py
   ```

### 에이전트가 시작 안 될 때
1. "에이전트상태"로 현재 상태 확인
2. 스크립트 경로 확인:
   - 데이브: `skills/데이브_트레이더/tools/upbit_auto_trader.py`
   - 레오: `skills/레오_공격형트레이더/tools/leo_aggressive_trader.py`
   - 펄스: `skills/펄스_전략가/tools/crypto_market_intelligence.py`

### 409 Conflict 에러
- 다른 곳에서 봇이 실행 중
- 해결: "봇전체종료" → 10초 대기 → "봇시작"
