# 장전 브리핑 (Morning Brief — KRX)

> Anthropic `equity-research/morning-note`를 한국시장·소미 도구에 맞게 재작성. 미국 7am 모닝미팅이 아니라 **한국 08:30 장전 브리핑** 포맷이다.

## 데이터 소스 (하드코딩 금지 — 항상 실시간)

- 관심종목: `output/cache/somi_watchlist.json` (고정 종목 없음 — watchlist가 대상 결정)
- 수급·점수·현재가: `somi_kis_reporter.py` / KIS API
- 시장 레짐: `market_regime.py` (위험선호/중립/위험회피)
- 급등락·거래량 급증: `somi_price_monitor.py` 최근 알림
- 보유 포지션: `somi_position_monitor.py`
- 뉴스/이벤트: `market_trend_alert.py`, LLM(`_shared/llm.py`)

## 워크플로

### 1단계 — 야간·전일 점검
- **미국장 마감**: 3대 지수·나스닥·반도체(SOX)·환율(USD/KRW)·야간선물. ADR 상장 한국기업(쿠팡 등) 동향.
- **관심종목 오버나잇**: 관련 미국 동종업체 급등락, 공시·뉴스, 대차/공매도 변화(`short_covering_analyzer.py`).
- **시장 레짐**: `market_regime.py` 결과를 한 줄로 — 오늘 공격/중립/방어 중 어디인가.

### 2단계 — 브리핑 포맷 (2분 컷)

```
📈 [YYYY-MM-DD] 소미 장전 브리핑

🔑 오늘의 한 줄: [PM이 꼭 알아야 할 단 하나]
- 핵심 전개 2~3문장 + 시장 레짐(공격/중립/방어)

🌙 야간·해외
- 미국장: 지수/반도체/환율 한 줄 + 우리 시장 함의
- [종목A]: 한 줄 이벤트 + 소미 코멘트
- [종목B]: 한 줄 + 코멘트

📅 오늘 일정
- [시간] 실적발표/IR
- [시간] 경제지표(컨센 vs 소미 뷰)
- 수급 이벤트(옵션만기·MSCI 리밸런싱·배당락 등)

🎯 매매 아이디어 (있으면)
- [매수/관망/감시] [종목]: 1~2문장 논리 + 트리거 가격
- 무효화 조건(틀렸다고 인정할 지점)

📌 보유 포지션 액션: 익절/손절/홀딩 — somi_position_monitor 결과 반영
```

### 3단계 — 관심종목 빠른 점검 표

| 종목 | 소미점수 | 수급(외/기) | 현재가/등락 | 코멘트·액션 |
|------|---------|-----------|-----------|-----------|
| | | | | 관망/감시/매수후보 |

### 4단계 — 출력 (사용자 규칙 준수)
- **텔레그램**: 위 압축본 전송(`_shared/notify.send`). 주식 질문엔 항상 분석 + 요약 텔레그램 (사용자 규칙).
- **노션**: 길어지면 풀버전은 노션, 채팅/텔레그램엔 핵심만 (사용자 규칙 [[report-to-notion]]).
- 1페이지 초과 금지.

## 중요 노트
- **뷰가 있어야 한다** — 뉴스 요약만 하고 판단이 없으면 쓸모없다.
- "특이사항 없음"도 유효한 브리핑 — "오버나잇 무재료, 포지션 유지"라고 명시.
- 액션(매수/관망/익절/손절)은 **모의=자동, 실거래=승인 필요** 원칙([[paper-mode-autotrade]])을 따른다 — 브리핑은 제안까지, 실거래 체결은 별도 승인.
- 06:30 작성 시 "장 시작 시점에 바뀔 수 있음" 타임스탬프 명시.
- 모든 수치는 KIS/실데이터 출처 — 추정이면 `[추정]` 표기, 하드코딩 금지([[no-hardcoding]]).
