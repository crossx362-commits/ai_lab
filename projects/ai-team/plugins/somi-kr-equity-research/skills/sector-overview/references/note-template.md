# 섹터 분석 — 노트 템플릿 (노션 풀버전)

상위 도구표: [`../../references/TOOLCHAIN.md`](../../references/TOOLCHAIN.md).

```markdown
# [섹터/테마] 섹터 리포트 — [YYYY-MM-DD]

## 1. 요약 (TL;DR)
- 사이클 위치 / 수급 방향 / 톱픽 3종목 (3~5줄)

## 2. 시장 개요
- 전방 수요·글로벌 사이클(반도체/전기차 등), 정책·수출
- 밸류체인: 소재 → 부품 → 장비 → 완성 (마진 귀속 지점)

## 3. 경쟁 구도
| 종목(코드) | 시총 | 소미점수 | 수급(외/기) | PER/PBR | 차별점 |
|-----------|-----|---------|-----------|---------|-------|

## 4. 밸류에이션
- 섹터 PER/PBR 밴드(현재 vs 과거), KOSPI/KOSDAQ 대비 프리미엄/디스카운트

## 5. 투자 함의 & 톱픽
- 톱픽 3종목: 한 줄 논리 훅 + 트리거 + 무효화 조건
- 강세/약세 논쟁, 섹터 내러티브 촉매(실적·정책·증설)

## 6. 리스크
- 테마 과열·유동성·정책 리스크

## 출처
- KIS/공시/데이터 출처. 추정 [추정].
```

## 도구
- 종목 수급: `somi_kis_reporter.make_report()`
- 후보 채점: `somi_screener.run()` → 톱픽은 `idea-screen`으로 정밀 채점
- 텔레그램엔 TL;DR + 톱픽만, 본문은 노션([[report-to-notion]]).
