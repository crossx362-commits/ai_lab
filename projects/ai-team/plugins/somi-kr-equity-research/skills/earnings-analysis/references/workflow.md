상위 도구표: [`../../references/TOOLCHAIN.md`](../../references/TOOLCHAIN.md).

## 한국시장 특수성
- **영업이익**이 핵심 지표(EPS보다 우선). 잠정실적 → 확정공시 2단계.
- 컨센서스(시장 추정치) 없으면 `[컨센 미상]` 표기.
- 일회성(자산매각·충당금·환율) 제거한 **본업 수익력**을 분리.

## 절차
1. **수집**: DART/공시·KIS에서 매출·영업이익·순이익(YoY/QoQ).
2. **서프라이즈 표**: 컨센 대비 매출·영업이익·영업이익률·순이익.
3. **질적 분석**: 서프라이즈 원인(물량/단가/믹스/환율) 지속성, 가이던스·컨콜.
4. **수급·주가 반응**:
   ```python
   from somi_kis_reporter import make_report   # 발표 후 외인/기관 매매
   from short_covering_analyzer import flow_short_analysis  # 공매도·대차 변화
   ```
   "실적 vs 주가" 괴리(선반영/미반영) 판단.
5. **투자논리 영향**: `thesis-tracker`의 어느 기둥을 강화/약화하나 → 액션.
6. **출력**: 텔레그램 한 줄 서프라이즈 + 소미 뷰 + 액션 / 노션 풀 리뷰.
