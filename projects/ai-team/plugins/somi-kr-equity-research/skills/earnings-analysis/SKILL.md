# 실적 분석 (Earnings Analysis — KRX)

> Anthropic `equity-research/earnings-analysis`를 한국 회계·공시(잠정실적·DART) 기준으로 재작성. 핵심 지표는 **영업이익**(한국시장 핵심)이며 EPS는 보조.

## 워크플로

### 1단계 — 실적 수집
- DART/공시·KIS API에서 분기 매출·영업이익·순이익(잠정/확정)
- 컨센서스(시장 추정치) 확보 — 없으면 `[컨센 미상]` 표기
- 전년동기(YoY)·전분기(QoQ) 비교

### 2단계 — 서프라이즈 표

| 지표 | 컨센서스 | 실적 | 서프라이즈 | YoY | QoQ |
|------|---------|-----|----------|-----|-----|
| 매출 | | | | | |
| 영업이익 | | | | | |
| 영업이익률 | | | | | |
| 순이익 | | | | | |

### 3단계 — 질적 분석
- **서프라이즈 원인**: 물량/단가/믹스/환율/일회성 — 지속 가능한가?
- **가이던스/코멘트**: 상향/하향/유지, 컨콜·IR 핵심
- **세그먼트**: 사업부별 동인
- **재무 건전성**: 재고·매출채권 추세, 현금흐름

### 4단계 — 수급·주가 반응
- 발표 후 외국인·기관 매매 방향(`somi_kis_reporter.py`)
- 갭/거래량, 공매도·대차 변화(`short_covering_analyzer.py`)
- "실적 vs 주가" 괴리 — 선반영/미반영 판단

### 5단계 — 투자논리 영향 & 액션
- 이 실적이 투자논리(`thesis-tracker`)의 어느 기둥을 강화/약화시키나?
- **액션**: 관심종목 유지 / 비중확대 후보 / 트림 / 회피
- 모의=자동, 실거래=승인([[paper-mode-autotrade]]).

### 6단계 — 출력
- 텔레그램: 서프라이즈 한 줄 + 소미 뷰 2~3문장 + 액션.
- 노션: 풀 리뷰(표·세그먼트·컨콜 요약) ([[report-to-notion]]).

## 중요 노트
- 한국시장은 **영업이익 서프라이즈**에 민감 — EPS보다 우선.
- 잠정실적은 세부 미공개 — 확정 공시에서 업데이트 필요 명시.
- 일회성(자산매각·충당금) 제거한 본업 수익력을 분리해서 볼 것.
- 모든 수치 출처 명시, 추정 `[추정]`, 하드코딩 금지([[no-hardcoding]]).
