# 종목 발굴 (Idea Screen — KRX)

> Anthropic `equity-research/idea-generation`을 한국시장·소미 점수 체계로 재작성. 미국식 P/E·SaaS 스크린이 아니라 **한국 수급·거래대금·소미 점수·기대값** 기반이다.

## 데이터 소스

- 후보 풀: `somi_screener.py` (거래대금 상위), KIS API
- 채점: `somi_signal_engine.py` (소미 점수), `short_covering_analyzer.py` (대차/공매도)
- 시장 레짐: `market_regime.py` — 레짐에 따라 스크린 강도 조절
- 매수구간: `somi_trade_advisor.py`

## 워크플로

### 1단계 — 탐색 기준 정의
사용자에게(또는 기본값으로) 확인:
- **방향**: 매수 후보 / 관망 감시 / 회피
- **시장**: 코스피 / 코스닥 / 전체
- **시총대**: 대형 / 중형 / 소형 (코스닥 소형은 변동성 경고)
- **스타일**: 수급주도(외인·기관 순매수) / 거래대금 급증 / 숏커버링 / 테마
- **시장 레짐 반영**: 위험회피 레짐이면 후보 수·비중 축소

### 2단계 — 스크린 (소미 점수 체계)

**수급주도 스크린**
- 외국인·기관 동반 순매수(N일 연속)
- 거래대금 전일比 급증 + 가격 상승 동반
- 대차잔고 감소 / 공매도 비중 하락(숏커버링 단서)
- 프로그램 순매수 유입

**거래대금 모멘텀 스크린**
- 거래대금 상위 + 신고가 근접
- 거래량 급증(평균 대비 배수) + 양봉
- 섹터 동반 강세(같은 테마 복수 종목)

**숏커버링 스크린** (`short_covering_analyzer.py`)
- 높은 공매도 비중 + 대차잔고 급감
- 펀더멘털 악재 부재 + 수급 반전

**회피/숏감시 스크린**
- 외국인·기관 동반 순매도 지속
- 거래대금 동반 없는 급등(작전 의심)
- 관리종목·불성실공시·감자/유증 리스크
- 대차잔고 급증 + 거래량 둔화

### 3단계 — 채점 & 기대값
각 후보를 소미 점수로 채점하고, 가능하면 **기대값(EV) = (상승확률 × 목표수익) − (하락확률 × 손절폭)** 으로 정렬([[somi-expected-value-engine]]).

| 종목 | 소미점수 | 수급 | 거래대금 | 기대값(EV) | 매수구간 | 한 줄 논리 |
|------|---------|-----|---------|-----------|---------|----------|

### 4단계 — 숏리스트 출력
- 상위 3~5종목, 각 **한 줄 논리 훅 + 트리거 가격 + 무효화 조건**.
- 텔레그램 요약 전송, 상세는 노션([[report-to-notion]]).
- 관심종목 미등록 종목이면 "관심종목 추가 <코드> <명>" 안내(`watchlist_manager.py`).

## 중요 노트
- 데이터는 전부 KIS 실시간 — 종목/수치 하드코딩 금지([[no-hardcoding]]).
- 코스닥 소형주·테마주는 변동성·유동성 경고를 반드시 병기.
- 발굴은 **후보 제안까지**. 실제 매수는 모의=자동/실거래=승인([[paper-mode-autotrade]]).
- 스크린이 0건이면 "오늘 조건 충족 종목 없음 — 관망"이 정답. 억지로 만들지 말 것.
