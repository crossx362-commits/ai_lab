# 📊 AI 에이전트 리포트 저장소

이 폴더는 AI 에이전트들이 생성하는 리서치 보고서, 학습 로그, 작업 히스토리를 저장합니다.

## 📁 폴더 구조

### 1. `research/` - 리서치 보고서
에이전트들이 수행한 리서치 결과물을 저장합니다.

**저장되는 파일:**
- 아린 (관리자): 인스타그램 트렌드 리서치
- 현빈 (전략가): 비즈니스 리서치, 시장 분석
- 루나 (디렉터): YouTube 트렌드, 음악/영상 리서치
- 기타 에이전트들의 리서치 보고서

**파일 형식:**
- JSON: 구조화된 리서치 데이터
- Markdown: 상세 리포트 문서

### 2. `learning/` - 학습 로그
에이전트들의 학습 과정과 성능 개선 데이터를 저장합니다.

**저장되는 파일:**
- `success_log.jsonl`: 성공 사례 (보상 학습)
- `fail_log.jsonl`: 실패 사례 (패널티 학습)

**목적:**
- 에이전트 성능 향상
- 오류 패턴 분석
- 학습 데이터 축적

### 3. `history/` - 작업 히스토리
에이전트들의 작업 기록과 실행 내역을 저장합니다.

**저장되는 파일:**
- `upload_history.json`: 콘텐츠 업로드 기록
- 기타 작업 로그

## 🔄 자동 저장 규칙

### 리서치 보고서
- **저장 경로**: `reports/research/{agent_name}_research.json`
- **저장 시점**: 리서치 작업 완료 시
- **파일명 형식**: `{에이전트명}_research_{날짜}.json`

### 학습 로그
- **저장 경로**: `reports/learning/success_log.jsonl` (성공) / `fail_log.jsonl` (실패)
- **저장 시점**: 작업 완료 후 성과 평가 시
- **형식**: JSONL (줄단위 JSON)

### 작업 히스토리
- **저장 경로**: `reports/history/{작업명}_history.json`
- **저장 시점**: 주요 작업 실행 시
- **보관 기간**: 최근 30일

## 📝 파일 명명 규칙

```
{에이전트명}_{작업타입}_{날짜}.{확장자}

예시:
- arin_research_20260602.json
- luna_trend_analysis_20260602.md
- hyunbin_market_research_20260602.json
```

## 🗂️ 파일 관리

### 자동 정리
- 30일 이상 된 학습 로그는 아카이브
- 중복 리서치 보고서는 최신 버전만 유지
- 히스토리는 최근 100개 항목만 보관

### 백업
- 주요 리서치 보고서는 별도 백업
- 학습 로그는 주간 단위 압축 백업
