#### 고위험
1. **클라이언트 사이드 프리미엄 검증** (`freemium.js:58-62`)
   ```js
   if (params.get('premium') === 'activated') {
       localStorage.setItem("petna_premium", "stripe_verified");
   ```
   → URL 파라미터 조작으로 누구나 무료 프리미엄 획득 가능
   → **패치:** Stripe webhook → Supabase `premium_users` 테이블 기록 → 서버 검증

   ```js
   const apiKey = window._env_?.GEMINI_API_KEY || "";
   ```
   → `window._env_`가 클라이언트 JS에 노출되면 Gemini API 키 탈취 가능
   → **패치:** Vercel Edge Function 프록시로 API 키 서버 사이드 보관

3. **localStorage 기반 상태 신뢰** (`state.js:AppStore`)
   - 게임화 데이터(IQ점수, 조화도)가 localStorage에만 저장
   - 유저 직접 조작 가능 → 소셜 기능의 신뢰성 훼손

### 패치 필요 항목 (우선순위 순)
- 항목1: GEMINI_API_KEY → Vercel Edge Function 프록시 (`/api/gemini-health`)

## 👑 예원 CEO — 최종 결정

### 오늘 회의 핵심 결론
코드베이스 분석 결과, 펫과나는 **기능은 풍부하나 3개의 치명적 약점**이 있다:
① 프리미엄 수익 모델이 보안 취약점으로 무력화 가능 (즉시 패치 必)
② AI 날씨 데이터가 가짜라 사용자 신뢰 훼손 (시한폭탄)
③ 조화도/사주 탭이 최고 차별점인데 UX가 열악 (서브탭 없이 세로 스크롤만)

경쟁사(TTcare, 펫피) 대비 **사주+MBTI+소셜 결합**이 유일한 차별점이므로, 이를 강화하면서 기술 기반을 먼저 다진다.

### 즉시 구현 TOP 3 (우선순위 순)

1. **조화도 탭 서브탭 네비게이션**: 최고 차별점인 사주 탭이 UX 문제로 사장되고 있음. social.js의 switchSocialSubTab 패턴 그대로 복사 적용으로 1시간 내 완성 가능. 예상 임팩트: 조화도 탭 체류시간 300% 증가, 이탈률 감소.

2. **날씨 실제 API 연결**: 현재 fake 날씨는 유저가 1번 써보면 앱 신뢰를 잃게 만드는 시한폭탄. OpenWeatherMap 무료 티어(분당 60호출)로 충분. 예상 임팩트: 마이펫 탭 재방문율 향상.

3. **오늘의 운세 카드 인스타 공유**: 바이럴의 핵심. share-card.js 기반이 있고 saju.js 데이터도 있어 연동만 하면 됨. 매일 아침 유저가 카드를 공유하면 신규 유저 유입 루프 형성. 예상 임팩트: 유기적 설치 전환 월 +30%.
