# 🔐 환경변수 보안 규칙

## ⚠️ 핵심 원칙
**모든 환경변수는 암호화하여 최상위 루트 한 곳에 보관**

## 📂 중앙 집중식 관리

### 위치
```
d:/ai_lab/.env (암호화된 중앙 파일)
```

### 접근 방법
```python
from _shared.env_loader import load_env
load_env(PROJECT_ROOT)  # 자동 복호화 후 로드
```

## 🚫 금지 사항
- ❌ 프로젝트별 개별 .env 파일 생성
- ❌ 환경변수 하드코딩
- ❌ GitHub에 .env 커밋
- ❌ 평문 환경변수 저장

## ✅ 모든 에이전트 규칙
1. **load_env(PROJECT_ROOT) 필수 호출**
2. **os.getenv()로만 환경변수 접근**
3. **환경변수를 직접 파일에 작성 금지**
4. **중앙 .env 파일만 사용**
