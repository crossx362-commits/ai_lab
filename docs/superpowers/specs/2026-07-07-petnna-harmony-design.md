## 목표
조화도 탭을 "측정하고 끝"에서 **리포트급 결과 + 매일 올 이유 + 펫게임 시너지**로 확장.

## 1. 명리 계산 코어 (harmony-core.js) — 오너 승인분
- **년주**: `(연도-4)%10`→천간, `(연도-4)%12`→지지. **월지**: 월별 지지(인월=2월 기준 표준 매핑: 1월=축…12월=자, 입춘 무시 간이). **일간지**: 기준일(1900-01-01=갑술일) 대비 일수 % 60 순환.
- 천간→오행: 갑을=목 병정=화 무기=토 경신=금 임계=수 / 지지→오행: 인묘=목 사오=화 진술축미=토 신유=금 해자=수
- **오행 벡터**: [년간, 년지, 월지, 일간, 일지] 5글자 집계 → {목,화,토,금,수} 카운트
- **궁합(0~100)**: 두 벡터의 원소쌍(5×5=25쌍) 중 상생(목→화→토→금→수→목) 비율×40 가산, 상극(목⊣토, 토⊣수, 수⊣화, 화⊣금, 금⊣목) 비율×25 감산, 동일오행 쌍 비율×10 가산, 기저 50, 클램프 5~99.
- **영역별 궁합 4종**: 놀이=화, 식사=토, 산책=목, 휴식=수 주관. `영역점수 = 궁합×0.5 + (펫 주관오행 강도 + 집사 주관오행 강도)×보정 + 상생보너스`, 5단계 등급(최고/좋음/보통/노력/주의) × 영역별 팁 문구(20개 테이블 내장).
- **일진**: 날짜별 일간지의 오행을 펫 오행 벡터와 대조. 길일 카테고리: 산책=일진이 목·화이고 펫 대표오행과 상생, 미용=금 일진, 병원=수·토 일진 중 상생일. **오늘의 궁합 지수** = 기본 궁합 + 일진 보정(상생 +15/상극 -15/중립 0), 클램프 5~99.
- **저장 호환**: 결과는 기존 `pet.harmonyData` 확장 — 기존 필드(`avgScore` 등) 유지 + `elements:{pet,owner}`, `areas:[4]`, `dominant:{pet,owner}`, `measuredAt` 추가. 마이룸 위젯·소셜 공유(`avgScore` 소비) 무수정 호환.
- **UMD-lite**: `window.PetHarmony` + `module.exports` (petgame 패턴 동일), node 테스트 가능.

## 2. UI (harmony-view.js) — 자율 확정분
- **영혼 조화도 탭**: 측정 결과(`harmony-result-container`) 아래 `#harmony-plus` 컨테이너에:
  ① **오행 분포 차트** — 5행(목화토금수) × 펫/집사 페어 가로 막대(div 기반, SVG 불요). 색: 목 #7BA05B·화 #E05C5C·토 #E8A55A·금 #B9A88C·수 #5C8FC9. 각자 대표 오행 뱃지("댕쿠는 물(水)의 기운").
  ② **상생상극 요약** — 대표 오행 관계 1줄 해석(상생 5문구·상극 5문구·동일 1문구 테이블).
  ③ **영역별 궁합 2×2 카드** — 놀이🎾/식사🍚/산책🐾/휴식💤: 점수·등급·팁.
  ④ **오늘의 실천 가이드** — 오늘 일진 기반 1문구.
- 렌더 시점: 측정 완료 직후 + 저장된 harmonyData 있으면 탭 진입 시 자동. 미측정 시 `#harmony-plus` 비움.
- **일진 캘린더(오늘의 운세 탭)**: `fortune-test-section` 상단 `#fortune-calendar`에 이번 달 그리드 — 길일 이모지 마킹(🐾✂️🏥), 오늘 하이라이트+오늘의 궁합 지수 배너, 월 이동(전/다음). 미측정 시 "조화도 먼저 측정하기" CTA 버튼(→harmony 탭 전환).
- 크림+코랄 팔레트(DESIGN.md), 기존 탭 스타일(rounded-2xl·border-amber/rose-100) 준수.

## 3. 게임 버프 — 자율 확정분
- `game-core.feed()`: `PetHarmony.todayIndex(pet)` ≥ 70이면 **XP ×1.2**(반올림). typeof 가드로 하모니 모듈 부재 시 무영향.
- `game-core.decay()`: `pet.harmonyData` 존재 시 감쇠 **×0.8** (행복 유지 보상).
- HUD(game-stage): 버프 활성 시 코인 옆 `💞1.2x` 미니 뱃지.
- 밸런스 근거: 하루 먹이 XP 상한이 낮아(수십) 1.2배는 진행 가속이지 인플레 아님.
